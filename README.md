## 1. ❗ 서비스 소개

- 세탁해병대는 바쁜 현대인을 위한 비대면 세탁 대행 서비스. 인터넷으로 신청해서 수거 및 배송을 손쉽게 집에서 편하게 이용할 수 있는 사이트. 세탁뿐만 아니라 보관, 수선 서비스까지 이용 가능.
